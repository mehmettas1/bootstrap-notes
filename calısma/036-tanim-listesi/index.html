Tanım Listesi
Tanım listesi, terimlerin açıklamalarıyla birlikte listelendiği listeleme türüdür. <dt> etiketi terimleri, <dd> etiketi ise açıklamaları tanımlar.
HTML
HTML, web sayfaları ve uygulamaları oluşturmak için kullanılan işaretleme dilidir.
CSS
CSS, bir işaretleme diliyle yazılmış belgenin sunumunu tanımlamak için kullanılan bir stil sayfası dilidir.
Javascript
Javascript, web sayfaları ve web tabanlı uygulamalarda etkileşimli yapılar oluşturmanızı sağlayan bir betik dilidir.
Bootstrap
Bootstrap, web sayfaları ve web tabanlı uygulamaların ön yüzlerini geliştirmek için kullanılan bir uygulama geliştirme çatısıdır.
